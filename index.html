The Decentralized Blockchain Graveyard

DeGrave.io is creating a multi-chain ancestor memorializing platform that will be fulfilled by incorporating Binance Smart Chain and other compatible blockchains.

Whitepaper

Our Mission

Our mission is to provide an easily accessible ancestors memorializing system for users despite time and space constraints. Our project is a blockchain graveyard where users can create NFT graves, collate deceased social media info, purchase digital offerings and make donations.

Our Solution

The DeGrave.io ecosystem is configured to meet the requirement in an innovative, robust and user-friendly manner. We are creating a blockchain ancestors memorialize system where users can create NFT graves, digitalized memory collections and collate social media’s information whilst allowing information to be transferred among stakeholders securely. Ancestors’ worshippers can use Deathcoin to purchase offerings and make donations.

Features

5 fantastic features for achieving excellent user experience

Users Verification

All users will be verified with ID checking using KYC, and death cert checking for launching NFT grave. We will configure an AI capability to perform the verification process in the future.

Blockchain Graveyard


DeGrave.io creates a non-fungible GRAVE with carrying values as the physical grave. We build an ecosystem around it in which values are held. We will create an intangible, environmentally friendly and space-saving Blockchain Graveyard.

Memorializing System

Allow users to purchase digital offerings and make donations to the bereaved with records on the monumental page of the NFT grave. Show your respect and love in the comment section.

Memory Collections

Create media albums and collate social media links to preserve the precious memories with your beloved ones. Media content can also be created as NFTs’ memorable collections to trade on the platform.

Reward System

Get your Deathcoin reward daily by launching your verified NFT grave. Maximize your reward through participating in the Deathcoin community to drive forward the adoption of DeGrave.io protocols

NFT Specification

  • Conform to the ERC1155 specification
  • Metadata makes use of the same format and rules specified by all leading NFT compatible platforms
  • Each of NFT’s metadata contains a signature signed by DeGrave.io’s system’s private key

Content Storage

  • Stored on the IPFS network
  • Metadata is stored as a JSON file on IPFS
  • The hash of the JSON metadata file is stored on chain as the token URI
  • Integrity of the NFT as a whole is guaranteed
  • Now: files stored on our IPFS cluster
  • Future: store files on one of the external IPFS service providers
